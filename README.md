# Brighthouse.News.Api


# Standard .Net Api Solution

The project was created using the normal Visual Studio template.
Currently using .net 10, so please download and install the the .net 10 sdk
The database is sqllite and it has been attached to the project
The migrations have already been applied.
No AI tools were used to develop any features, everything is hand written, but google was used to assist with certain issues.

## Run the app
Just hit F5 to run the api solution and you can browse to the swagger definition 
https://localhost:7121/swagger/index.html
